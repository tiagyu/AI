1. 데이터셋 개요
- 본 데이터셋은 2019년 ~ 2021년 연합뉴스 데이터를 제공받아 구축되었습니다.
- 2개의 뉴스 기사를 하나의 페어로 하여 제목, 본문 문장, 이미지 캡션을 swap 하는 방식을 통해 생성된 제목과 본문이 다른 가짜뉴스 탐지 데이터셋 입니다.

2. 데이터 구성 요소
- 본 데이터셋은 'title', 'subtitle', 'body', 'caption', 'headline_id', 'body_id', 'category', 'classcode', 'label' 컬럼으로 구성되어 있습니다.
- ‘label’ 값은 다음과 같습니다.
	- 0: 진짜 뉴스
	- 1: 가짜 뉴스

3. 데이터 분할
- 본 데이터셋은 train (194,860개) / valid (10,826개) / test(10,826개) 로 구성되어 있습니다.
